# Booking Sorft

Sistema de gestión hotelera desarrollado como proyecto formativo SENA.

---

## Descripción

Booking Sorft es una solución orientada a la administración de hoteles, permitiendo gestionar reservas, clientes, habitaciones y procesos internos de forma eficiente.
